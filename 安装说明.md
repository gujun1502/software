# Out Building Decision · 安装与使用说明

招投标商机雷达 + 决策引擎。每天自动读采招网订阅邮件 → 只挑设计类 → 用你的资质/业绩打分排序 → 出 PDF 决策日报 + 推送微信。

---

## 一、它怎么工作（一张图）

```
采招网订阅邮件 → IMAP自动读取 → 解析项目 → 只留"设计"类(剔除工程/施工/供货)
   → 决策引擎按你资质+业绩打分排序 → PDF决策日报 + 微信TOP8推送
```

每天 9:30 / 9:45 / 10:00 三个时间点自动跑（哪个时刻电脑开着就由最早的跑，当天只跑一次）。

---

## 二、核心文件

| 文件 | 作用 |
|---|---|
| `daily_job.py` | 每日总调度（带"当天只跑一次"保护）|
| `fetch_email.py` | IMAP 读采招网邮件 |
| `parse_email.py` | 解析邮件、分类、判定是否设计类 |
| `decision_engine.py` | 决策打分（硬门槛 + 业绩契合度）|
| `run_radar.py` | 主流程：解析→打分→出日报→推送 |
| `push_wechat.py` | 微信文字推送（PushPlus/Server酱）|
| `scripts/md_to_pdf_edge.py` | 跨平台 PDF 生成（自动找 Chrome/Edge）|
| `company_profile.json` | 你的资质 + 业绩库 |
| `email_config.json` | 邮箱 IMAP 配置（授权码）|
| `push_config.json` | 微信推送配置（token）|

---

## 三、Windows 安装

1. 装 Python 3（python.org），勾选 "Add to PATH"。
2. 装依赖：命令行运行 `pip install requests beautifulsoup4 lxml markdown`
3. 手动跑：双击 `每日商机雷达.bat`
4. 自动跑：定时任务已建（任务名 `OutBuildingDecision_DailyRadar`）。

---

## 四、macOS（Mac Mini）安装 ⭐

> Mac Mini 常年开机，最适合做这个每天自动跑的"服务器"。

1. **拷贝**：把整个 `OutBuildingDecision` 文件夹拷到 Mac（比如放到"文稿"里）。
2. **装 Python 3**（若没有）：python.org 下载，或终端 `brew install python`。
3. **装 Chrome**（生成 PDF 用）：https://www.google.com/chrome/
4. **一键安装**：
   - 打开"终端"(Terminal)，把 `install_mac.command` 拖进终端窗口，回车；
   - 或：双击 `install_mac.command`（若提示"无法打开"，右键→打开→打开）。
   - 它会自动装依赖、配好每天 9:30/9:45/10:00 的定时任务。
5. **手动跑一次试试**：双击 `每日商机雷达.command`。

> ⚠️ 两台机器别同时跑（会重复推送）。装到 Mac 后，把 Windows 的定时任务关掉。

---

## 五、日常使用

- **每天**：电脑开着，到点自动出报告 + 推微信 TOP8，你啥都不用做。
- **想立刻跑**：双击 `每日商机雷达.bat`(Win) / `每日商机雷达.command`(Mac)。
- **看完整报告**：`reports\商机雷达日报_日期.pdf`。
- **改你的资质/业绩**：编辑 `company_profile.json`（业绩填得越全，排序越准）。
- **改关键词**：在采招网网站上改你的订阅关键词（它负责"找"）。

---

## 六、配置项（都在各自 json 里）

- `email_config.json`：`email`(邮箱) + `auth_code`(IMAP授权码)。
- `push_config.json`：`enabled`(true开启) + `token`(PushPlus) 或 `sendkey`(Server酱)。
- `company_profile.json`：资质、业绩库、打分权重。

> 这些文件含你的账号凭证，**只存在本机、勿外传**；凭证可随时在对应网站重置。

---

## 七、出问题怎么办

- 没出报告 / 没收到推送：看 `data/daily_log.txt`(Win) 或 `data/launchd.log`(Mac) 的报错。
- 卸载 Mac 定时：终端 `launchctl unload ~/Library/LaunchAgents/com.obd.dailyradar.plist`
